TMAP 이동경비 자동 작성 HUB

이동 양식, 방문활동관리, 근태관리, 카드승인내역을 넣으면 자가용/대중교통 경비 양식에 자동 기입합니다.

사용 순서: ① 엑셀 4개 업로드 → ② 미리보기 생성 → ③ 완성본 다운로드
TMAP 앱키는 Vercel 환경변수 TMAP_APP_KEY에 넣어야 하며, 키가 없거나 호출 실패 시 이동 양식 DB의 거리/통행료를 보조값으로 사용합니다.

1. 이동 양식

자가용/대중교통 시트가 들어있는 원본 양식 .xlsx

2. 방문활동관리

방문날짜, 방문시간, 매장명이 있는 파일

3. 근태관리

근무일자, 출근지점, 퇴근지점이 있는 파일

4. 카드승인내역

승인일자, 가맹점명, 승인금액, 취소여부가 있는 파일
TMAP 상태 확인 전

진행 상태

파일을 업로드하고 “미리보기 생성”을 눌러주세요.
