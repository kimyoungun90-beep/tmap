# TMAP 이동경비 자동 작성 HUB

일반 엑셀 파일 4개를 업로드해서 자가용/대중교통 경비 양식에 자동 입력하는 웹앱입니다.

## 처리 흐름

1. 이동 양식.xlsx 업로드
2. 방문활동관리 파일 업로드
3. 근태관리 파일 업로드
4. 카드승인내역 파일 업로드
5. 미리보기 생성
6. 완성본 엑셀 다운로드

## 자동 입력 기준

### 자가용 이용 업무 수행 경비 사용내역서

- A열: 날짜
- B열: 출발지
- D열: 도착지
- F열: TMAP 또는 양식 DB 기준 운행거리
- J열: TMAP 또는 양식 DB 기준 적정통행료
- M열: 카드내역 중 주유소/충전소 결제금액 날짜별 합산
- N열: 통행료

C/E/I/K/L열의 기존 수식은 최대한 유지합니다.

### 대중교통 이용 업무 수행 경비 사용내역서

카드내역에서 아래 항목을 찾으면 대중교통 시트에 입력합니다.

- SR
- SRT
- KTX
- 코레일
- 한국철도
- 가맹점업종: 철도

입력 위치:

- A열: 날짜
- B열: 기차
- C열: 행선지 자동 추정
- D열: 업무 이동
- F열: 공급가액
- G열: VAT
- H열: 기존 수식 유지

## TMAP 앱키 설정

Vercel 프로젝트에서 환경변수로 등록하세요.

```bash
TMAP_APP_KEY=발급받은_TMAP_앱키
```

주의: 앱키를 `app.js`, `index.html`에 직접 넣지 마세요. 브라우저 소스에 노출됩니다.

## 배포 방법: Vercel 기준

1. 이 폴더를 GitHub 저장소에 업로드
2. Vercel에서 Import Project
3. Project Settings > Environment Variables
4. `TMAP_APP_KEY` 추가
5. Deploy

## 로컬 실행

Vercel CLI가 필요합니다.

```bash
npm install
npm run dev
```

## 좌표 수정

`place-db.js`에서 점포/사무실 좌표를 수정할 수 있습니다.

특히 `대구 사무실`은 현재 임시 좌표입니다. 실제 사무실 주소/좌표로 수정하면 이동거리 정확도가 좋아집니다.

```js
{
  name: '대구 사무실',
  lon: 128.6138,
  lat: 35.9063
}
```

## TMAP 미연결 시 동작

TMAP 앱키가 없거나 호출이 실패하면, 이동 양식의 `DB` 시트 G:J열에 있는 거리/통행료를 보조값으로 사용합니다.
